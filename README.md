# RacingCar
## Description

Just a fun racing game!

## Controls
- W to accelerate
- S to reverse 
- A/D to direction car
- ESC to close window

## DEBUG keys

- F1 to free camera
- F2 Aerial Cam
- F3 Side Cam
- F4 Free Cam
